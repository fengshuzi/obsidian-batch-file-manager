# Batch File Manager

一个强大的 Obsidian 批量文件管理插件，支持批量选择、删除、移动文件。

## 功能特性

- ✅ 批量选择文件（全选/取消全选）
- ✅ 批量删除文件
- ✅ 批量移动文件到指定文件夹
- ✅ 显示选中文件数量统计
- ✅ 右键菜单快速操作
- ✅ 点击文件名快速打开
- ✅ 文件列表按路径排序

## 安装

### 开发模式安装

1. 将此插件文件夹复制到你的 Obsidian vault 的 `.obsidian/plugins/` 目录下
2. 在插件文件夹中运行：
   ```bash
   npm install
   npm run build
   ```
3. 在 Obsidian 设置中启用 "Batch File Manager" 插件

### 快速开发

```bash
cd obsidian-batch-file-manager
npm install
npm run dev
```

## 使用方法

1. 点击左侧边栏的文件图标，或使用命令面板搜索 "打开批量文件管理器"
2. 在右侧面板中会显示所有 Markdown 文件
3. 勾选要操作的文件
4. 使用工具栏按钮进行批量操作：
   - **全选**: 选中所有文件
   - **取消全选**: 取消所有选中
   - **删除选中**: 批量删除选中的文件（不可撤销）
   - **移动选中**: 批量移动文件到指定文件夹
   - **刷新**: 重新加载文件列表

## 快捷操作

- 点击文件名：打开文件
- 右键文件：显示快捷菜单（打开、删除）
- 勾选复选框：选择/取消选择文件

## 注意事项

- 删除操作不可撤销，请谨慎使用
- 移动文件时会自动创建目标文件夹
- 如果目标位置已存在同名文件，移动操作会失败
